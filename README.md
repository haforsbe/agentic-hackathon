![Python](https://img.shields.io/badge/python-3.10%2B-blue.svg)

Hands-on examples for building agents with the Agent Framework and Microsoft Foundry.

## Quickstart

### 1) Create and activate a virtual environment

Windows (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

macOS/Linux:

```bash
python -m venv .venv
source .venv/bin/activate
```

### 2) Install dependencies

```bash
pip install -r requirements.txt
```

### 3) Run examples

```bash
python 01-agent-framework/01_hello_agent.py
python 02-microsoft-foundry-agents/01-quickstart-responses.py
```

## Project structure

- `01-agent-framework/`: Core Agent Framework samples
- `02-microsoft-foundry-agents/`: Foundry agent quickstarts
- `03-microsoft-foundry-hosted-agents/`: Hosted agent scenarios and workflows
- `Prereqs*.md`: Setup prerequisites and environment guidance
